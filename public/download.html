◆◆◆ 7 Diamonds – הורדת קבצים לגיטהאב

כל הקבצים של הפרויקט – לחצי על כל קובץ כדי לראות ולהעתיק

📋 איך מעלים לגיטהאב

  1. פתחי github.com וצרי ריפו חדש: 7-diamonds
  2. לחצי על Add file → Create new file
  3. הדביקי את שם הקובץ (כולל תיקייה, למשל css/style.css)
  4. לחצי על העתק ליד הקובץ שלמטה ← הדביקי בגיטהאב
  5. לחצי Commit changes – חזרי על כל הקבצים
  6. לאחר מכן הפעילי GitHub Pages מ-Settings → Pages

🚀 הפעלת GitHub Pages

1
היכנסי לריפו ב-GitHub → Settings
2
גללי לחלק Pages בתפריט הצדדי
3
תחת Source בחרי Deploy from a branch → main → / (root)
4
לחצי Save – אחרי כמה דקות האתר יהיה חי בכתובת https://USERNAME.github.io/7-diamonds
✅ הועתק ללוח!
